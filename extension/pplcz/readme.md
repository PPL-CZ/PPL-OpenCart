# PPL Plugin pro OpenCart 4.* (pre-release version)

Snadné a efektivní řešení dopravy pro váš e-shop na OpenCart.

PPL Plugin pro OpenCart umožňuje jednoduchou integraci přepravních služeb PPL do vašeho obchodu. Zákazníkům nabídne rychlé doručení, pohodlný výběr výdejních míst a přehledné sledování zásilek. Administrátorům zase usnadní práci automatizací procesů a rychlou tvorbou štítků.

---

## Hlavní vlastnosti

### Tvorba štítků
Automatické generování přepravních štítků přímo z administrace OpenCart bez ručního přepisování dat.

### Výběr doručovacích metod
Podpora doručení na adresu, PPL ParcelShop, ParcelBox a dalších služeb PPL.

### Sledování zásilek (Track & Trace)
Automatické přiřazení sledovacího čísla a možnost zobrazit aktuální stav zásilky.

### Automatické ověření adresy
Validace a doplňování adres pomocí PPL API pro snížení chybovosti objednávek.

### Hromadné zpracování objednávek
Vytváření štítků a odesílání dat pro více objednávek najednou.

### Dynamický výpočet ceny dopravy
Automatická kalkulace ceny podle hmotnosti, rozměrů nebo cílové destinace.

### Podpora dobírky (COD)
Možnost platby na dobírku s automatickým zpracováním částky.

### Notifikace zákazníků
Zaslání e-mailového nebo SMS upozornění zákazníkovi o změně stavu zásilky.

### Podpora více skladů
Konfigurace různých výchozích adres při expedici z více skladových míst.

### Jednoduché nastavení
Intuitivní konfigurace v administraci OpenCart v sekci **E**
